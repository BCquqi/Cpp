图的存储: 邻接矩阵 邻接表
邻接矩阵: 空间O(n^2)
    1. 概念
        无权图/带权图(有权图)
        无向图对角线对称(有向图不对称)
        无权图: 可用bool二维数组
        带权图: 应用int/LL二维数组, 如果有0权可以设置为+-无穷(看做最短路还是最长路)
        邻接矩阵不可以存储重边! 实在需要使用邻接矩阵, 按照题目要求取舍重边
    2. 常见用法
        有矩阵G[][], 如何找到x的邻点?(01)
        一般dfs时间复杂度O(n + m)
邻接表: 只存储存在的边(一般使用vector存储)
    1. vector(向量, 矢量) -- 动态数组
        a. 定义: vector<int> G; // 此时已经是一维
                vector<int> G[N]; // 此时是二维
                vector<Edge> G; // 结构体vector, 一般用来附加权值
        b. 增加: G[i].push_back(j); // 点i存在一条边指向j
        c. 删除: G[i].pop_back(); // 图论一般不用
        d. 修改: G[i][j] = x;
        e. 查询: tmp = G[i][j];
链式前向星(易出错, 一般不允许STL时使用)
欧拉路径与欧拉回路: 
    1. 欧拉路径: 从一个点出发, 经历所有边后, 到达v
    2. 欧拉回路: 从一个点u出发, 经历所有边后, 回到u
    3. 欧拉路径存在的充要条件: 有0个或2个奇度点 2奇度点一定1起点1终点
    4. 欧拉回路存在的充要条件: 没有奇度点

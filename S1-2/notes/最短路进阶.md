# 最短路

| | 算法 | 时间复杂度 | 空间复杂度 | 限制 |
| :---: | :---: | :---: | :---: | :---: |
| 多源最短路 | Floyd | $O(n^3)$ | $O(n^2)$ | |
| 单源最短路 | BFS | $O(n + m)$ | $O(n + m)$ | 无权图 / 等权图 |
| | SPFA | $O(n \times k)$ | $O(n \times k)$ |
| | Dijkstra | $O(n^2) \to O(m \times \log n)$ | $O(n^2) \to O(m \times \log n)$ | 无负权图 |
